Data used in this project (CPRM)
  - historical (2000 to 2019)
      1. 1hr PrAdjust (precipitations, spatial res: 2.5km, temp res: 1hr)
      2. day tasAdjust (median temperature, spatial res: 2.5km, temp res: day)
  - 1st projection scenario (2042 to 2061)
      1. 1hr PrAdjust (precipitations, spatial res: 2.5km, temp res: 1hr)
      2. day tasAdjust (median temperature, spatial res: 2.5km, temp res: day)
  - 2nd projection scenario (2068 to 2087)
      1. 1hr PrAdjust (precipitations, spatial res: 2.5km, temp res: 1hr)
      2. day tasAdjust (median temperature, spatial res: 2.5km, temp res: day)

  Time and material constraints, as well as data errors, limited our ability to download and process all data (for example, corrupted file for precipitations for the year 2042).
  Scripts have only been run for precipitations on the 2042-2061 period as of writing this.
